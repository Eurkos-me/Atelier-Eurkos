Back Office PC Configuration

Section 1 - Prep Work
1.1 - Applicable Tools

The following tools will be used during setup of a Back Office PC

  1. Cisco Jabber - used to call the store and/or vendor tech.
  2. IVY - used to assign a static network host address to the PC. Also known as "locking down".
  3. UltraVNC - used to connect to the PC remotely to perform the setup steps.
  4. PingInfoView - used to watch the dhcp range for a new device coming online.
1.2 - On-site Requirements

The following criteria will need to be met in the store in order to perform Back Office PC setup.

  1. Store personnel (or vendor technician) available to install new device.
  2. Device located.
  3. Device plugged into power.
  4. Device plugged into network.
  5. Keyboard, mouse, and monitor plugged in.
Section 2 - Configuration
2.1 - Lock Down PC

Use IVY to assign the new device a static network host name.

  1. Open PingInfoView and enter dhcp01.st0###.st.coat.com to find the IP address for that store's dhcp01 host name.
  2. Enter that IP address in instead of the full FQDN, and then add a hyphen and the same IP address, but with .16 as the last octet instead of .1
    • Example: dhcp01.st00681.st.coat.com returns an IP address of 10.11.205.1, so you would enter 10.11.205.1-10.11.205.16 into PingInfoView.
  3. Have the vendor tech (or store associate) power on the device and watch for one of the listed DHCP addresses to come online in PingInfoView.
  4. Use that host address to connect to the device in UltraVNC and verify with the vendor tech that it is the correct device.
  5. Open a command window on the device and enter the following command:
    getmac
  6. Open IVY and log in with your SSO credentials.
  7. Use the top search bar to search for the store in question
  8. Find the host name that you're going to be locking the device in question down with, and enter the device's MAC address into the host name entry.
  9. In the command window on the device, enter the following command:
    ipconfig /release && ipconfig /renew
  10. If the device don't refresh its connection to use the static host instead of the dhcp host, have the vendor tech reboot the device.
2.2 - Personalization

Personalization is the part of the setup that involves setting up this device as a specific PC at a specific store, rather than a clone of a generic device.

  1. Through VNC, open an elevated command prompt and enter: c:\scripts\setup.bat
  2. Use the form below to determine how to name the PC.
